# githubrepos-api

Simple GitHub API repository search app
Enter your topic and find the repositories with this topic in its name!

---

# Tecnologies

Languages: TypeScript

Tools: Vite

---

# How to run

- Pre-requisites: npm

```bash
npm install
npm run dev
```

---
